# Summary

<h3>Very Customizable Fidget Spinner</h3>
<h4>Customize:</h4>
<ul>
<li><em>type of weights</em>: 5/16", 3/8", 1/2" nut, M8, M10, M12 nut, 608 bearing, 5/16", 3/8", 1/2", 5/8" loose ball bearing, US penny, nickel, dime or quarter, M4 square nut</li>
<li>number of rays</li>
<li>number of weights per ray</lli>
<li>edge roundness (radius)</li>
<li>tapering of rays</li>
<li>glue grooves</li>
</ul>
<h4>Update 2017-05-03</h4>
added M4 square nuts as suggested by <a href="http://www.thingiverse.com/nattrassj12/about">nattrassj12</a>
<h4>Update 2017-04-25</h4>
<ul>
<li>added the option to use 5/16" (8mm) loose ball bearings</li>
<li>added the option for horizontal glue grooves for center 608 bearing and ray weights</li>
</ul>
<h4>Update 2017-04-11</h4>
added the option to use loose ball bearings: 3/8", 1/2", and 5/8" (tested with 1/2" and 5/8"); for a heavy, long spinning spinner. Thanks to <a href="http://www.thingiverse.com/Stoli666/about">stolli666</a> for the suggestion.
<h4>Update 2017-03-27</h4>
added option to taper rays or not; without tapering, the ray will be more rectangular and less triangular
<h4>Update 2017-03-02</h4>
<ul>
<li>added M10 and M12 nut sizes</li>
<li>fixed low resolution bug</li>
</ul>
<h4>Update 2017-02-23</h4>
<ul>
<li>added 3/8" and 1/2" nut choices for weights</li>
<li>nuts will protrude but didn't want to require supports or have an asymmetrical frame</li>
</ul>
<h4>Update 2017-02-15</h4>
<ul>
<li>added the ability to extend the ray length when using only 1 weight per ray</li>
<li>added coins to type type of weights: 4 quarters, 5 dimes, 3 nickels or 4 pennies. But I prefer nuts or bearings for weights.</li>
</ul>
