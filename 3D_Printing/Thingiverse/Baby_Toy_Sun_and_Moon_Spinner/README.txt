# Summary

I really liked caj's idea, but i though it would be more fun to switch between a sun and a moon. also changed the two halves of the frame from screw connection to superglue, but added locating pins to make sure they are oriented properly before the glue sets.

before gluing the sphere together, i added a few beads to make it rattle too.
